How to use Blend
  1. Set your main image with Upload or the Sources strip — that is the bottom layer.
  2. Use Choose image or Use result here for the top layer. Status shows Ready when the top layer is set.
  3. This row appears when Blend is selected in the effects bar.
  4. Adjust Mode and Opacity; the preview updates in the result panel.
