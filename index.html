How to use Blend
  1. Use npm run dev for hot reload; API runs on port 3000.
  2. Set your main picture (Upload / Sources) — that is the bottom layer.
  3. Here, choose Choose image or Use result — that is the top layer (status shows Ready).
  4. This row appears only when Blend is selected in the effect bar above.
  5. Change Mode and Opacity here; the result updates on the right.
